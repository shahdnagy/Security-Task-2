# 🔐 Cryptography Algorithms in Java

## 📖 Overview
A Java-based project that implements a collection of classical and modern cryptographic algorithms.  
The goal is to demonstrate how encryption and decryption techniques work in a clear and practical way.

---

## 🧩 Implemented Algorithms

### 🔸 Classical Ciphers
- Caesar Cipher
- Monoalphabetic Cipher
- Playfair Cipher
- Hill Cipher
- Rail Fence Cipher
- Columnar Transposition
- AutoKey Cipher
- Repeating Key Cipher

### 🔸 Modern Ciphers
- AES (Advanced Encryption Standard)
- DES (Data Encryption Standard)

---

## ⚙️ Features
- 🔁 Supports both encryption and decryption  
- 🧠 Covers multiple cryptography techniques  
- 📚 Clean and simple implementations for learning purposes
